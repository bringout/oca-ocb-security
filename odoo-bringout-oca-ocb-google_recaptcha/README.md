# Google reCAPTCHA integration


        This module implements reCaptchaV3 so that you can prevent bot spam on your public modules.
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-google_recaptcha
```

## Dependencies

This addon depends on:
- base_setup
